# SiteSense-AI-A-RAG-Powered-Website-Question-Answering-Chatbot

Built a Streamlit-based RAG application that scrapes websites, chunks text, generates embeddings, and performs semantic search using FAISS with multi-query chat support.

Integrated TinyLlama-1.1B with an end-to-end retrieval pipeline using BeautifulSoup, Sentence-Transformers, FAISS, and Transformers to generate accurate, context-aware answers.

INSTALL LIBRARIES:
pip install requests beautifulsoup4 sentence-transformers faiss-cpu streamlit transformers accelerate torch einops protobuf
